------------
插件提供；
	ctrl + alt + n 新建文件
	shift + delete 删除文件（自己绑定的）

------------
多行游标：
	1 ctrl + d：选中光标所在文本，继续按则选下一个
	  ctrl + k,ctrl + d：跳过刚刚选中的，并选择下一个
	2 alt+F3：选所有
	3 ctrl + shift + l：选中区域每行后面加光标
	4 shift +鼠标右键拖动
	5 ctrl +alt+↑/↓ 向上、下添加多行光标
	按esc可以取消多行游标状态

------------
啥都能跳转：
	ctrl + p 模糊匹配 有预览
	/ 路径
	@ # 
	: 跳转到行数

------------
emmet:
	ctrl + e：和tab功能一致
	{}中是内容： h2{hello}*10

------------
编辑：
	ctrl + enter 在下一行插入新行
	ctrl + shift + enter 在上一行插入新行
	ctrl + j 选中代码合为一行
	ctrl + shift + d 复制光标所在整行，插入到上一行
	ctrl + k + k 从光标处开始删除至行尾
	ctrl + shift + k 删除整行
	ctrl + F2 设置书签
	ctrl + t 左右字母互换
	ctrl + m：在起始、结尾括号间切换 () [] {}
	ctrl + shift +m 选择括号中的内容 () [] {}
	ctrl + l：选择整行，按住继续选择下一行
	shift + ↑/↓ 向上、下选中多行
	shift + ←/→ 向左、右选中文本
	ctrl + shift + ↑/↓：移动当前行
	ctrl + l 选中整行
	ctrl + ← 向左单位性地移动光标
	ctrl + → 向右单位性地移动光标
	ctrl + shift + ← 向左单位性地选中文本
	ctrl + shift + → 向右单位性地选中文本
	ctrl + shift + ↑ 将光标所在行和上一行代码互换（将光标所在行插入到上一行之前）
	ctrl + shift + ↓ 将光标所在行和下一行代码互换（将光标所在行插入到下一行之后）

缩进：
	左缩进： tab  			ctrl + [
	右缩进： shift + tab  	ctrl + [

折叠：
	ctrl + shift +[/] 折叠、展开代码
	ctrl + k + 0 展开所有折叠代码

显示：
	ctrl +tab 按文件浏览过的顺序，切换当前窗口的标签页。
	ctrl + pagedown/pageup 向左/右切换当前窗口的标签页
	f11 全屏
	shift +f11 免打扰
