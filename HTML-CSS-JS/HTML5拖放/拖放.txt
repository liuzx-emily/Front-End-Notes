IE8-不支持拖放

-----------
	*draggable*
为了使元素可拖动，把draggable属性设置为true
	<div draggable="true">

-----------
	*拖放事件*
1 被拖的元素
	dragstart drag dragend
2 拖入的元素
	dragenter dragover dragleave drop
3 元素默认不能被drop。只有在dragover事件中阻止默认事件，才能在该元素上drop，也就是才能触发drop事件。不能释放、能释放的光标不一样。
	oDiv.ondragover = function(ev) {
	    ev.preventDefault();
	};
4 事件执行顺序：
  1 drop不触发的时候：
	dragstart - drag - dragenter - dragover - dragleave - dragend 
  2 drop触发的时候：
	dragstart - drag - dragenter- dragover - drop - dragend

-----------
	*FF下的问题*
FF下默认只能拖拽图片。想要拖拽非图片，需要在该元素的dragstart中设置dataTransfer对象


-----------
	*dataTransfer对象*
1 方法
  1 setData(key,value) key和value都必须是string类型
  	ev.dataTransfer.setData('id','div1');
  2 getData(key) 根据key获取value
  	alert(ev.dataTransfer.getData('id'));
  3 setDragImage(元素，坐标x，坐标y) 以px为单位
  	ev.dataTransfer.setDragImage(oImg, 10, 10);
2 属性
  1 effectAllowed 设置“可以drop时的”光标样式
	none copy copyLink copyMove link linkMove move all uninitialized
  2 files 获取外部拖拽的文件，返回一个filesList列表 是类数组对象
	filesList下有个type属性，返回文件的类型
	（ajax实现无刷新上传文件，用的也是files对象）

	*实例：拖到垃圾桶里删除

-----------
	*拖拽外部文件*
1 因为被拖文件是外部，所以没有“被拖元素”。只需要给"拖入元素"设置事件。
2 拖入元素事件：dragenter dragover dragleave drop
3 dragover事件中：ev.preventDefault();
	为了能drop
  drop事件中：ev.preventDefault();
  	拖拽外部文件，鼠标抬起时，浏览器会默认打开该文件。要想不打开，需要阻止默认事件。
4 ev.dataTransfer.files：获取外部拖拽的文件，返回一个filesList列表
	ev.dataTransfer.files[0].type type属性返回文件类型
  

-----------
	*FileReader对象*
1 readAsDataURL方法：参数为要读取的文件对象，将文件读取为DataUrl 
2 load事件：当读取文件成功完成的时候触发此事件
	this.result 获取读取的文件数据
例子：
	oDiv.ondrop = function(ev) {
        ev.preventDefault();
        var fs = ev.dataTransfer.files;
        var fr = new FileReader();
        fr.readAsDataURL(fs[0]);
        fr.onload = function() {
            alert(this.result);
        };
    };
