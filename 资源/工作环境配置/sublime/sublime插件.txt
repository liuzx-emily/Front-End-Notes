2 SideBarEnhancements
	一键预览html
	安装好插件后，需要在keybindings中加入：

		（快捷键、地址自己改）
		{ "keys": ["f3"], "command": "side_bar_files_open_with",
	         "args": {
	            "paths": [],
	            "application": "D:/Firefox火狐浏览器/firefox.exe",
	            "extensions":".*"
	        },
	增强侧边栏：导航栏中project->add folder to project

3 Color Highlighter
	显示颜色+取色（快捷键ctrl+shift+c）
	在user文件中设置：

		{
		  "ha_style": "filled",
		  "icons": false
		}

	代码块注释
	可以快速的对函数进行注释。保存代码规范

	/**:回车在自动查找函数中的形参等等。
	它会生成 JSDoc 格式的注释。

13 AutoPrefixer:(依赖Node.js)	【必备】
	浏览器私有属性前缀补全，对CSS和SCSS都行 
	修改快捷键：在keybinds中添加：

		{ "keys": ["ctrl+alt+p"], "command": "autoprefixer" },

	修改user设置：详见
	http://www.w3cplus.com/css3/autoprefixer-css-vender-prefixes.html）

14 SublimeLinter
	修改csslint，使“css属性不按字母表顺序排列”时不报错：
		在sublimeLinter设置文件中csslint下：改成
		"ignore": "order-alphabetical",
		具体可查看https://github.com/SublimeLinter/SublimeLinter-csslint
	
	某个新版本？提示改的，在user中："tooltips": true
