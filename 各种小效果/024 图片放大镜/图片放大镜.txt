-----------
	总体效果
最开始小遮罩是display:none
鼠标在oSmall上时，小遮罩display:block
鼠标不在oSmall上时，小遮罩display:none
  代码：
	oSmall.onmouseenter = function(ev) {
	    ev = ev || window.event;
	    oMagnifier.style.display = 'block';
	};
	oSmall.onmouseleave = function(ev) {
	    ev = ev || window.event;
	    oMagnifier.style.display = 'none';
	};

鼠标在oSmall上移动时：
	1 小遮罩移动
	2 大图移动
  代码：
	oSmall.onmousemove = function(ev) {
	    ev = ev || window.event;
	    var L = ev.clientX - oSmall.offsetLeft - oMagnifier.offsetWidth / 2;
	    var T = ev.clientY - oSmall.offsetTop - oMagnifier.offsetHeight / 2;
	    if (L < 0) {
	        L = 0;
	    } else if (L > oSmall.offsetWidth - oMagnifier.offsetWidth) {
	        L = oSmall.offsetWidth - oMagnifier.offsetWidth;
	    }
	    if (T < 0) {
	        T = 0;
	    } else if (T > oSmall.offsetHeight - oMagnifier.offsetHeight) {
	        T = oSmall.offsetHeight - oMagnifier.offsetHeight;
	    }
	    oMagnifier.style.left = L + 'px';
	    oMagnifier.style.top = T + 'px';

	    var scaleX = L / (oSmall.offsetWidth - oMagnifier.offsetWidth);
	    var scaleY = T / (oSmall.offsetHeight - oMagnifier.offsetHeight);
	    oImgBig.style.left = scaleX * (oBig.offsetWidth - oImgBig.offsetWidth) + 'px';
	    oImgBig.style.top = scaleY * (oBig.offsetHeight - oImgBig.offsetHeight) + 'px';
	};

-----------
	问题：子级影响父级
例子：
    #div1 {width: 200px;height: 200px;background: red;}
    #div2 {width: 100px;height: 100px;background: yellow;}

	<body>
	    <div id="div1">
	        <div id="div2"></div>
	    </div>
	    <script>
	    var oDiv = document.getElementById('div1');
	    oDiv.onmouseover = function() {document.title += '1';};
		oDiv.onmouseout = function() {document.title += '2';};
	    </script>
	</body>
1 鼠标从"oDiv外界空白"移到"内层div2"时：
	效果：title加上"1"
	原因：冒泡
2 鼠标从"内层div2"移到"oDiv外界空白"时：
	效果：title加上"2"
	原因：冒泡
3 鼠标从"外层oDiv"移到"内层div2"时：
	效果：title加上'21'
	原因：鼠标离开外层oDiv了，所以title加"2"
		  内层div2的"鼠标进入事件"冒泡到外层oDiv，所以title加"1"
所以纯“阻止冒泡”不能解决问题
	给内层div2的mouseover和mouseout都阻止冒泡
	对情况3：title不会加1了，但还是会加2

解决：
	用onmouseenter onmouseleave(子级不会影响到父级)

-----------
	计算小图中的“半透明小遮罩”位置
L = ev.clientX - oSmall.offsetLeft - oMagnifier.offsetWidth / 2;
说明：
	1 减去oSmall.offsetLeft只在当前这种布局适用:
		1 没有滚动条，所以没有滚动条距离scrollLeft
		2 小图的定位父级就是oSmall，而oSmall就是相对与document定位的
	2 减去oMagnifier.offsetWidth / 2是为了让鼠标位置在小遮罩中间
	【如果有滚动条，或者定位父级情况变了，则L的表达式必须修改】

然后作位置判断：
	if (L < 0) {
	    L = 0;
	} else if (L > oSmall.offsetWidth - oMagnifier.offsetWidth) {
	    L = oSmall.offsetWidth - oMagnifier.offsetWidth;
	}
最后赋值：
	oMagnifier.style.left = L + 'px';


-----------
	计算“大图”位置
x方向的：
	小遮罩最大偏移：oSmall.offsetWidth - oMagnifier.offsetWidth
	大图最大偏移：oBig.offsetWidth - oImgBig.offsetWidth
js代码：（L是处理过的）
	var scaleX = L / (oSmall.offsetWidth - oMagnifier.offsetWidth);
	oImgBig.style.left = scaleX * (oBig.offsetWidth - oImgBig.offsetWidth) + 'px';
